# Zelo

**Zelo** is a web platform to learn **programming logic and fundamentals** interactively.  
Designed for beginners, it has short lessons, quizzes, and interactive exercises.

## MVP Features

- Dashboard with 3 columns: Progress / Learning Path / Actions
- Dynamic lesson nodes (clickable)
- Basic layout with React + Vite

## Tech

- React (JS)
- Vite
- HTML & CSS

## Run Locally

```bash
git clone https://github.com/your-username/zelo.git
cd zelo
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
